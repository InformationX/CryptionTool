# CryptionTool
![](./assets/img/UCT.gif)

**简介:** 一个CTF+渗透测试工具框架，主要实现功能在加解密和编码这块，在软件使用过程中发现问题或建议欢迎提交 issue.

**简介:** 不定期

---

# 目前支持的功能

**工具**
- 网络工具
  - Port Scan(端口扫描)
- 文字处理
  - Case conversion(大小写转换)
  - Text substitution(文字替换)

**加解密/编解码**
- 现代
  - 认证
    - NTLM-HASH
    - JWT
  - AES(Test)
  - Hash
    - MD5
    - MD5-16
    - MD2
    - MD4
    - SHA-1
    - SHA-224
    - SHA-256
    - SHA-384
    - SHA-512
  - SM3
- 传统
  - Atabash(埃特巴什码)
  - ROT13
  - Rail Fence(栅栏密码)
  - Vigenere(维吉尼亚密码)
- 编码
  - ASCII
  - Base64
  - Base Conversion
  - HEX
  - Morse Code
  - URL
  - Unicode

**待实现...**
- DES,3DES
- SM2,SM4
- LM
- 更多的文本处理
